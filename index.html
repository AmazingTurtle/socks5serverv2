A socks5 server in c# (v2)

Download as .zip Download as .tar.gz View on GitHub

Socks 5 server implementation in CSharp

RFC reference: https://www.ietf.org/rfc/rfc1928.txt

Functional working (asynchronous) SOCKS5 server written in C#. Supports both IPv4 and IPv6.

  • Connect command works fine (TCP)
  • Udp associate is in progress
  • Bind command is not planned to be supported, but maybe on demand
  • Supports plugins
  • Database connection (login, stats, billing, services)
  • Statistics

Functional service website is planned
